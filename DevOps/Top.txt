== Global ==
Z colors
B bold
E/e summary memery scale task memery scale

== Toggle Summary ==
l load average
t task /cpu stat
m memory /swap info

== Toggle ==
'0' zeros; 
'1/2/3' cpus or numa node views; 
'I' Irix mode
'R' Sort; 
'H' Threads; 
'V' Forest view; 
'J' Num justify
'c' Cmd name/line; 
'i' Idle; 
'S' Time; 
'j' Str justify
'z' color/mono; 
'b' bold/reverse (only if 'x' or 'y')

== Toggle highlights ==
 'x' sort field;  highlight the sorting fields
'y' running tasks

== Fields ==
'f'/'F' add/remove/order/sort; 
'X' increase fixed-width
Locate
'L'/'&' find/again; 
Move sort column: '<'/'>' left/right

== Filter ==
'u'/'U' effective/any user; 
'o'/'O' other criteria

== Set ==
'n'/'#' max tasks displayed; 
Show: Ctrl+'O' other filter(s)

== Manipulate ==
'k' kill; 
'r' renice
d or s   Set update interval
Write configuration file 'W'; 
Inspect other output 'Y'
