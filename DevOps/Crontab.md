====== Crontab ======
Created Saturday 30 August 2014

-l list
-e edit
-d delete
**contab M H D m d USER CMD**
M：分钟（0-59）
H：小时（0-23）
D：日期（1-31）
m：月份（1-12）
d：一个星期中的某天（0-6，0代表周日）
* (星号) null
or a file in etc/cron.d/sample.cron :LithuaniaLithuania
