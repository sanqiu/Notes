####unzip####
-  -d  ## to certain dir
-  -l  just list files not uncompress
-  -O chara code [GBK]
-  -t test
-  -r recursive
-  -q quiet
-  -9/1 compression level
-  -x exclude /file/name
-  [^.]*    ## include the .files
-  -t file.apk


###gzip:###

-  -d decompress
-  -f force
-  -k keep input files
-  -l list the compressed files ratio ,size
-  -v verbose
