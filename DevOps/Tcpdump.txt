====== Tcpdump ======
Created Friday 22 August 2014

tcpdump 'capture filter'
-D interface list
-i Listen on interface 
-l pipe it
-vvvvv
-w output to file
-r read from file
-ttt  delta in ms
-p  --no-promiscuous-mode ## tcpdump default to promiscuous mode
